# 堆
## 堆的两个特性
- 一个堆，已知节点个数，堆的结构一定是确定的，对于没有满的树层，先添加左节点，再添加右节点
- 一个堆，如果是最小堆，那么当前节点父节点的值一定小于等于当前节点的值。

## 特殊的实现
- [total_nums, 1, 2, 3, 4, 5]
- 对应的结构已经确定，node.left.index = node_index * 2, node.right.index = node_index * 2 + 1
